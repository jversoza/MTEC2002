Introduction

For this class, primarily as a way of uploading files to github

At the beginning of each class:

  1. open terminal
  2. change directory to Desktop {% highlight bash %} cd Desktop {% endhighlight %}
  3. make a directory with your first initial and last name {% highlight bash %} mkdir jversoza {% endhighlight %}
  4. change directory to the one that you just created {% highlight bash %} cd jversoza {% endhighlight %}
  5. log into github
  6. get the http URL of your repository
  7. make sure you're in the directory you just created {% highlight bash %} pwd # should return /Users/joe/Desktop/jversoza/ {% endhighlight %}
  8. download your assignments repository from github using the url that you copied above {% highlight bash %} git clone https://your_usrname@github.com/your_username/mtec2002_assignments.git {% endhighlight %}
  9. set your username (first initial, last name works) {% highlight bash %} git config user.name "your_username" {% endhighlight %}
  10. set your email {% highlight bash %} git config user.email "your_email@citytech.cuny.edu" {% endhighlight %}
  11. change directory to the repository that you just checked out {% highlight bash %} cd mtech2002_assignments {% endhighlight %}
  12. create a new directory for this class {% highlight bash %} # make sure to use the right number... we're using class 5 as an example mkdir class5 {% endhighlight %}
  13. change directory to the one that you just made {% highlight bash %} cd class5 {% endhighlight %}
  14. open that folder in finder
