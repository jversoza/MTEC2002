    About the language

    • Dynamic / interpreted programming language
    • Dynamically typed
    • Strongly typed
    • Whitespace significant
    • Multi-paradigm
    • Comprehensive standard library
    • Who has used Java and the java compiler? Technically it's compiled to bytecode, much like Java or c#, but there's no explicit compilation step
    • Processing/Java - explicit types. Dynamically typed - type checks are done at run time rather than compile time, allowing for for types and functionality to be generated at run time... allows duck typing (ex file-like classes in python)
    • Like java, but unlike JavaScript, PHP, etc... strongly typed (2 + "b")
    • How do you delimit code blocks in Processing/Java?
    • Imperetive, object oriented, and to some extent, functional programming
